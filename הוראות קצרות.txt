1. שימו את כל קבצי הבדיקות בתקייה אחת
2. באותה תקייה צרו קובץ חדש בשם self.py
3. העתיקו אל הקובץ הזה את הפונקציות שהתבקשתם לממש בתרגילים
4. פתחו חלון שורת פקודה באותה תקייה
5. הריצו את תוכנית הפייתון test_self.py

פעם אחת בלבד, יש צורך בהתקנת מודולים חיצוניים
הבדיקות הבסיסיות יכולות לעבוד גם בלעדייהם, אבל עדיף וקל להתקין אותם
בחלון שורת הפקודה הריצו:
python -m pip install -r requirements.txt
